# Sensors

Ultrasonic, IMU, temperature, encoders: how a robot perceives the world. Every autonomous behavior starts with a measurement.

## You can move on when you can...

- Wire and read a digital sensor (button, limit switch) and an analog one (potentiometer, photoresistor)
- Read a sensor over I2C or SPI and explain roughly what those buses are
- Explain sensor noise and apply a simple filter (moving average) to a reading
- Use an ultrasonic or time-of-flight sensor to measure distance
- Explain what an IMU measures and why raw gyro data drifts
