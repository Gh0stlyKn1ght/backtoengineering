# Motor Control

Servos, DC motors, steppers, and the driver boards that let a 3.3V pin command amps of current. Motion is what separates a robot from a sensor station.

## You can move on when you can...

- Sweep a hobby servo to a commanded angle from code
- Explain why a motor can't be driven straight from a GPIO pin
- Drive a DC motor forward/backward at variable speed with an H-bridge
- Explain the difference between servos, DC motors, and steppers, and when to use each
- Read an encoder and explain closed-loop vs open-loop control
