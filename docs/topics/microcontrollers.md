# Microcontrollers

Arduino, Raspberry Pi Pico, ESP32: tiny computers that read pins and run your code on bare metal. The bridge between the code you write and the circuits you build.

## You can move on when you can...

- Explain what a GPIO pin is and the difference between digital and analog I/O
- Flash a program onto a board (Arduino IDE or MicroPython) and see it run
- Read a button and drive an LED from code
- Explain PWM and why it's how you fake analog output
- Pick between an Arduino, a Pico, and an ESP32 for a given project
